# Lab 02 – Data Preprocessing and Visualization

## Aim

The purpose of this lab is to explore data preprocessing techniques and basic data visualization methods that help prepare datasets for machine learning applications. The exercises focus on detecting and handling missing values, cleaning and organizing datasets, modifying data types, and creating visual representations of data. These steps help in understanding patterns, trends, and relationships within the dataset before applying machine learning models.

---

## Dataset

The dataset used for this lab is included in this repository.

**File:** `data.csv`

This dataset is used to practice and demonstrate common data preprocessing tasks such as handling missing values, cleaning irregular or inconsistent entries, renaming columns for clarity, converting column data types, and generating visualizations to better understand the data.

---

## Libraries Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## Experiments

1. Identify missing values in a dataset.  
2. Remove rows containing missing values.  
3. Replace missing values using mean, median, or constants.  
4. Display and convert column data types.  
5. Rename column names for better readability.  
6. Identify and correct inconsistent values.  
7. Create line plots and bar plots.  
8. Create scatter plots between numerical features.  
9. Plot histograms to analyze data distribution.  
10. Create boxplots to identify outliers.
