# Lab 03 – Descriptive Statistical Analysis

## Aim

The objective of this lab is to perform descriptive statistical analysis on datasets in order to understand their basic characteristics before applying machine learning techniques. Using Python libraries such as NumPy, Pandas, SciPy, and Matplotlib, the experiments focus on calculating important statistical measures including mean, median, mode, variance, standard deviation, quartiles, correlation, and covariance. These techniques help summarize the dataset and reveal patterns or relationships among variables.

---

## Dataset

**Dataset Name:** Students Performance Dataset  

**Dataset Link:**  
https://github.com/rashida048/Datasets/blob/master/StudentsPerformance.csv  

This dataset provides information about students' academic results along with several background attributes that may influence performance. It is commonly used for data analysis tasks to examine how different demographic and educational factors relate to students’ scores in various subjects.

### Features in the Dataset

- gender  
- race/ethnicity  
- parental level of education  
- lunch  
- test preparation course  
- math score  
- reading score  
- writing score  

The dataset is often used in educational data analysis to explore trends in student achievement and investigate relationships between personal background and academic outcomes.

## Experiments

1. Compute measures of central tendency (mean, median, mode) and dispersion (min, max, variance, standard deviation).

3. Compute correlation and covariance between numerical variables.  
4. Visualize data distribution using histograms and boxplots.
